# Transformer based Predictive Maintenance for Textile Machinery
In this work, we use a lightweight transformer architecture for unsupervised time-series anomaly detection. Hence enabling predictive maintenance for automated embroidery machines in a small scale Textile MSME setup.

---

### Phase 1 : Data collection (IOT sensors inherent in the automatic embroidery machines to a central system for monitoring and analysis)
Time series anomally detection is used to identify unusual patterns or events in squential time-ordered data. This is an important problem across domains such as manufacturing, finance and healthcare. In particular, unsupervised time series anomaly detection has attracted significant attention due to the challenges of collecting abnormal data in real-world scenarios, such as the rarity of anomalies and the diversity of anomaly patterns.
